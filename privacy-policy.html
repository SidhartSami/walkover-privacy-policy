Privacy Policy

WalkOver is a walking activity tracking application developed by HighBrow Developers. This Privacy Policy explains how user data is collected, used, and stored.

Information We Collect

Location Data

WalkOver collects precise location data (GPS coordinates) only when the user explicitly starts a walking activity by using the “Start Walk” feature. Location tracking stops when the user ends the walk.

If a walk is not completed or the application closes unexpectedly, the walk data is not saved.

While location permission is granted, the app may display the user’s current location on the map, but this location is not stored unless an active walk is in progress.

Walk Data

When a walk is completed, the following data is saved:

  • Route path (latitude and longitude points)
  • Date and time of the walk
  • Total distance traveled

Authentication Data

Firebase Authentication is used only to allow users to sign in and access their own walking data. No authentication data is used for advertising or tracking purposes.

How We Use Your Data

Collected data is used solely to:

  • Display walking routes and history to the user
  • Calculate distance and activity statistics
  • Allow users to access their saved walks after signing in

Data Storage

Walk data and account information are securely stored using Firebase services.

Data Sharing

WalkOver does not sell, share, or rent user data to third parties. Data is not used for advertising or analytics.

Data Deletion

Users can delete individual walks within the app. When a walk is deleted, all associated data is permanently removed from our servers.

Children’s Privacy

WalkOver is not intended for children under the age of 13, and we do not knowingly collect personal data from children.

Changes to This Privacy Policy

This Privacy Policy may be updated from time to time. Any changes will be reflected on this page.
